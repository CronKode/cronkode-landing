# cronkode-landing

Official landing page for [CronKode (Pty) Ltd](https://cronkode.co.za) — a South African software company building vertical SaaS products for underserved African industries.

## Stack

- Plain HTML/CSS/JS — no frameworks, no build tools
- Hosted on Vercel
- Domain managed via Xneelo

## Structure

```
cronkode-landing/
├── index.html      # Main landing page
├── 404.html        # Custom 404 page
├── vercel.json     # Vercel deployment config
├── .gitignore
├── LICENSE
└── README.md
```

## Deployment

This repo deploys automatically to Vercel on every push to `main`. No build step required.

**Live URL:** https://cronkode.co.za  
**Preview URL:** https://cronkode-landing.vercel.app
